# LIMEN에 기여하기

LIMEN에 기여해주셔서 감사합니다! 이 문서는 기여 방법을 안내합니다.

## 기여 방법

1. **이슈 확인**: 먼저 기존 이슈를 확인하여 중복을 방지하세요.
2. **브랜치 생성**: 새로운 기능이나 버그 수정을 위한 브랜치를 생성하세요.
3. **코드 작성**: 명확하고 읽기 쉬운 코드를 작성하세요.
4. **테스트**: 변경 사항에 대한 테스트를 작성하고 실행하세요.
5. **Pull Request**: 변경 사항을 설명하는 PR을 생성하세요.

## 코딩 스타일

- **Go**: `gofmt`를 사용하여 코드를 포맷팅하세요.
- **TypeScript/React**: ESLint 규칙을 따르세요.
- **커밋 메시지**: 명확하고 간결한 커밋 메시지를 작성하세요.

## Pull Request 프로세스

1. PR 템플릿을 작성하세요.
2. 모든 테스트가 통과하는지 확인하세요.
3. 코드 리뷰 피드백에 응답하세요.
4. 변경 사항이 승인되면 머지됩니다.

## 질문이 있으신가요?

이슈를 생성하거나 기존 이슈에 댓글을 남겨주세요.
