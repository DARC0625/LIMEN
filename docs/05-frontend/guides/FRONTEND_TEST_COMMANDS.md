## ⚠️ 참고사항

이 문서는 과거 프론트엔드 개발 시 작성된 가이드입니다. 현재 LIMEN 프로젝트는 프론트엔드가 제거된 백엔드 전용 구조입니다. 향후 프론트엔드 재구축 시 참고용으로 보관됩니다.

---

## 네트워크 테스트

### ARP 캐시 정리

```bash
sudo ip -s -s neigh flush all
```

### 연결 테스트

```bash
# 백엔드 연결 확인
curl http://10.0.0.110:18443/api/health

# Agent 연결 확인
curl http://10.0.0.110:9000/health
```

### 포트 확인

```bash
# 포트 열림 확인
nc -zv 10.0.0.110 18443
nc -zv 10.0.0.110 9000
```
