# 프론트엔드 네트워크 설정

> [← 홈](../../00-home.md) | [프론트엔드](../) | [가이드](./) | [네트워크 설정](./FRONTEND_NETWORK_SETUP.md)

## ⚠️ 참고사항

이 문서는 과거 프론트엔드 개발 시 작성된 가이드입니다. 현재 LIMEN 프로젝트는 프론트엔드가 제거된 백엔드 전용 구조입니다. 향후 프론트엔드 재구축 시 참고용으로 보관됩니다.

---

## 네트워크 구조

```
프론트엔드 (10.0.0.10)
    ↓ 직접 연결
백엔드 (10.0.0.110:18443)
```

---

## 방화벽 설정

### 프론트엔드 서버

```bash
# 아웃바운드 허용
sudo ufw allow out to 10.0.0.110 port 18443 proto tcp
sudo ufw allow out to 10.0.0.110 port 9000 proto tcp
```

### 백엔드 서버

```bash
# 인바운드 허용
sudo ufw allow from 10.0.0.10 to any port 18443 proto tcp
```

---

## 연결 테스트

```bash
# 프론트엔드에서 실행
curl http://10.0.0.110:18443/api/health
```
