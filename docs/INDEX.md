## 📚 문서 업데이트 규칙

### 새 문서 추가 시
1. 적절한 파일명 사용 (영문, 하이픈 구분, 대문자 시작)
2. 이 INDEX.md에 링크 추가
3. 적절한 카테고리에 분류

### 기존 문서 수정 시
1. 변경 사항을 [CHANGELOG](./CHANGELOG.md)에 기록
2. 관련 Phase 문서 업데이트 (해당되는 경우)
