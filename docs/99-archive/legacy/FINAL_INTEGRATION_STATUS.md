- ✅ Envoy HTTP: 80 (외부 접근)

## 🔗 통신 흐름

```
외부 클라이언트
    ↓
Envoy (80/443) [프론트엔드 서버]
    ↓
프론트엔드 (10.0.0.10:9443)
    ↓
백엔드 (10.0.0.100:18443) [WSL]
Agent (10.0.0.100:9000) [WSL]
```

## 📋 방화벽 규칙

### 백엔드 서버 (WSL)
- 인바운드:
  - 10.0.0.10 → 10.0.0.100:18443 (백엔드)
  - 10.0.0.10 → 10.0.0.100:9000 (Agent)
  - SSH (22)

### 프론트엔드 서버
- 인바운드:
  - HTTP (80)
  - HTTPS (443)
  - SSH (22)
- 아웃바운드:
  - 10.0.0.100:18443 (백엔드)
  - 10.0.0.100:9000 (Agent)
  - DNS (53)
  - HTTPS (443) - 외부 API
