## 보안 원칙

### 1. 코드 분리
- 백엔드 서버에는 프론트엔드 코드 포함 금지
- 프론트엔드 서버에는 백엔드 코드 포함 금지
- RAG 문서만 공유 가능

### 2. 최소 권한 원칙
- 각 서버는 필요한 최소한의 코드만 포함
- 불필요한 파일 및 의존성 제거

### 3. 자동 검증
- 배포 시 자동 보안 검증
- 검증 실패 시 배포 중단
